### 注意点
- bzt 类产品列名存在编码差异：1-2月列名为 `{month}月营收{元）`，3-4月为 `{month}月营收（元）`，脚本两种格式都尝试匹配
- 数据服务类产品通过 `sjfw_module` 字段中的关键词在「模块」列中模糊匹配
- 全年收益采用滚动累加方式：上期全年收益 + 本月收益
- 上期 extract 文件不存在时，上月收益和全年收益返回 NaN
- 同比变化字段暂未实现，固定输出 `/`

### paragraphs_data_computed.py — 标桥段落计算

从标桥收益数据和 BP 表计算各项指标，追加到 `report_paragraphs.json`。

**数据源文件：**

| 文件 | 说明 |
|------|------|
| `source_data/营收平台标桥收益数据.xlsx` | 标桥收益数据（多 sheet），各 sheet 含「收益金额（元）」或「收益（元）」列 |
| `persistence_data/标桥_bp.xlsx` | 各分公司 BP 总额 |
| `process_data/extract_data{prior_month}月报.xlsx`「表格14」 | 上期数据，用于读取上月收益和全年收益 |

**逐键计算方式：**

| 键名 | 数据来源 | 计算逻辑 |
|------|----------|----------|
| 截至日期 | 当前年月 | `{year}.{month}.31` |
| 标桥本月营收总计 | 营收平台标桥收益数据 | 遍历所有 sheet，累加「收益金额（元）」或「收益（元）」列 |
| 标桥本月营收环比 | 本月 + 上期 extract | `(本月收益 - 上月收益) / 上月收益 × 100`，上月收益从上期 extract「表格14」的「本月收益（元）」列汇总 |
| 标桥本月营收同比 | — | 暂留空（需要去年同期数据） |
| 标桥全年营收总计 | 上期 extract + 本月 | 报告月为 1 月时 = 本月收益；其他月份 = 上期全年收益 + 本月收益（从上期 extract「表格14」的「全年收益（元）」列汇总） |
| 标桥全年BP总额 | 标桥_bp | `sum(BP总额（元）)` |
| 标桥全年BP完成率 | 计算值 | `全年营收总计 / BP总额`，格式化为百分比 |

### fill_docx_model.py — 模板填充

1. 读取 `model/运营中心营运产品收益月报_model.docx` 模板
2. 读取 `extract_data{month}月报.xlsx`，按 sheet 名映射到 docx 表格，逐行追加数据
3. 读取 `report_paragraphs.json`，替换段落中的 `{{占位符}}`
4. 输出到 `res_data/运营中心营运产品收益月报.docx`

### 执行顺序

```
paragraph_export.py        →  生成 report_paragraphs.json（提取段落）
paragraphs_data_computed.py →  追加到 report_paragraphs.json（计算区域市场化）
各 table_xx 脚本            →  生成 extract_data{month}月报.xlsx
fill_docx_model.py         →  读取 JSON + Excel，填充模板，输出最终月报
```
